= VS Code

== Shortcuts
* Open Command window to run vscode commands
** Ctrl+Shft+P
* Editor
** Ctrl+Shft+E
* Search
** Ctrl+Shft+F
* Source Control
** Ctrl+Shft+G
* Run
** Ctrl+Shft+D
* Extensions
** Ctrl+Shft+X
